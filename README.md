# CRT_FinalAssesment_SalesForceLogin
## Assumptions
- Salesforce login credentials are valid and the user has access to create Leads.
- Standard Lead fields such as First Name, Last Name, Company, Lead Source, Status, and Owner are available.
- Lead Status is assumed to be **Open** after successful Lead creation.
- Duplicate Lead warning, if displayed, is handled conditionally in the automation.
- Test data is generated dynamically to avoid hard-coded values.
- The automation is executed in a single Salesforce org and a single browser.
